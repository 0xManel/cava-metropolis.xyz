DESCRIPCION DEL PROYECTO
-------------------------
Cava Metropolis es una PWA (Progressive Web App) de gestion de carta/adega
multi-restaurante. Gestiona el catalogo de vinos de tres establecimientos:
SPA, Tasca Fina y Victoria, con BODEGA como catalogo corporativo principal.

Stack: HTML + CSS + JavaScript vanilla | JSON estatico | Vercel (deploy automatico)
URL en produccion: Vercel (auto-deploy en cada push a main)

ESTABLECIMIENTOS
----------------
1. BODEGA  — 2056 vinos con precio medio (100% cobertura) — CATALOGO PRINCIPAL
2. SPA     — 1080 vinos (52.5%)
3. TASCA FINA — 1311 vinos (63.8%)
4. VICTORIA   — 1195 vinos (58.1%)
5. GALERIA    — 0 vinos (futuro)

================================================================================
  WORKFLOW DE DEPLOY
================================================================================

  1. Editar archivos
  2. git add <archivo>
  3. git commit -m "tipo(scope): descripcion"
  4. git push origin main
  → Vercel detecta el push y hace deploy automatico

  Para forzar refresh en navegadores:
  → Cambiar version en sw.js (ej: 'stock-cava-2026-03-04-...')
