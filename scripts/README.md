# Scripts de desarrollo

Scripts Python para desarrollo y mantenimiento del proyecto.

## Archivos:

- **server.py** - Servidor local de desarrollo (Python HTTP)
- **generate_icons.py** - Genera todos los íconos PWA desde el logo
- **verify_pwa.py** - Verifica la configuración PWA
- **check_deployment.py** - Verifica que todo esté listo para deploy

## Uso:

```bash
# Iniciar servidor local
python3 scripts/server.py

# Generar íconos desde un logo
python3 scripts/generate_icons.py

# Verificar PWA
python3 scripts/verify_pwa.py

# Verificar deployment
python3 scripts/check_deployment.py
```
